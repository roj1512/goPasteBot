# goPasteBot

Go version of [dashezup/ezPasteBot](https://github.com/dahsezup/ezPasteBot).

## Requirements

- Golang
- A [Telegram bot token](https://t.me/botfather)

## Run

```
BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11 go run .
```
